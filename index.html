ℹ️ Running in demo mode — data is local to this device only. Connect Firebase so all 30 scouts share live data.

Firebase gives you a real-time database so finds, scores and caches sync across every phone instantly — for free at this scale.

  1. Go to console.firebase.google.com and create a project
  2. Choose Realtime Database → Start in test mode
  3. Project Settings → Your apps → Add web app → copy config
  4. Open this HTML file and paste your config into the <script type="module"> block at the top
  5. Replace the YOUR_* placeholders with your real values and reload
